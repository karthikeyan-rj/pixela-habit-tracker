# pixela-habit-tracker

A simple habit tracking app built with Python, Tkinter and the Pixela API. 
Log your daily progress and watch your consistency grow over time.

## Requirements

pip install requests tkcalendar


## How to Run

python main.py


## Features

- Create a new habit tracker with a custom username
- Log daily progress with a date picker
- Add, update or delete entries
- View your habit graph in the browser
- Login with saved credentials to continue tracking

## Built With

- Python 3
- Tkinter
- Pixela API
- tkcalendar
